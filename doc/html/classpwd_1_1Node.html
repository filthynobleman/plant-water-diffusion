Detailed Description

A class representing the node of a graph.

This class represents a node inside a graph structure.
It offers an interface for accessing node's properties and its relations with the graph.
Nodes represents oriented cylinders in space, hence they have an head, a tail and a radius.

Create a new node inside the given graph.

This constructor initializes a new node inside the given graph, using the given informations.
Notice the node is initialized completely disconnected to any other node, and connections must be set from the graph structure.
If the given graph is null, the constructor will throw a pwd::NullPointerException.
If head and tail are coincident, the constructor throws a pwd::AssertFailException.
Radius and mass must be strictly positive, otherwise the constructor throws a pwd::AssertFailException.

Parameters
GraphThe graph this node belongs to.

RadiusThe radius of the node.
IsOnLeafDetermine if this node is on a leaf area.
Exceptions
pwd::NullPointerExceptionif the graph is null.
pwd::AssertFailExceptionif head == tail or radius <= 0 or mass <= 0.
