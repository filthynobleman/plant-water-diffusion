Detailed Description

A class representing a graph structure.

This class represents a graph structure and exposes a set of methods for accessing the graph's properties and modifiyng it.
Notice that even if the structure is general, this graph actually represents a tree. Connections are undirected, the structure contains no cycles and has a root.

Creates a connection between two nodes.

This method creates a connection in the tree-graph between the node with ID ID1 and the node with ID ID2.
If one of the IDs does not belong to the tree-graph, the method throws a pwd::AssertFailException.
If ID1 == ID2 the method throws a pwd::AssertFailException.
If the connection already exists, the method does nothing.

Recompute head and tails of each node.

This method recomputes head and tail of each node in the tree-graph.
The head and tail recomputation occurs under the assumption that each node segment is connected to its neighbour.

std::unordered_map<const pwd::Node*, int> pwd::Graph::m_IDs
private

Map of the nodes' IDs.

This is a map from each node pointer to its ID.
Basically a reverse mapping of m_Nodes. Formally

m_Nodes[m_IDs[N]] == N
m_IDs[m_Nodes[i]] == i

for each valid node index i and valid node pointer N.
